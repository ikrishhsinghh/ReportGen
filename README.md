# ReportGen – Student Report Card Management System (Core Java)

## 📌 Project Overview

This project started as a **console-based Core Java application** developed for my Semester 2 syllabus (FY B.Sc. Computer Science).
Going beyond the syllabus, I upgraded the project on a personal level by integrating a **GUI using Java Swing**, transforming it into a desktop-style application while keeping all core logic and fundamentals intact.

The goal was to understand how real-world software evolves from simple command-line programs into user-friendly applications.

---

## 🚀 Features

* Admin login system
* Add student details
* Enter subject-wise marks
* Generate formatted report cards
* Modify existing student records
* Search students by name
* Display class-wise student performance
* Save and load data using file handling (no database required)
* GUI-based desktop interface using Java Swing
* Works completely offline (no internet needed)

---

## 🧠 Technologies & Concepts Used

* Core Java
* Object-Oriented Programming (OOP)
* Arrays & ArrayList
* File Handling & Serialization
* Exception Handling
* Java Swing (GUI)
* CMD to Desktop Application Upgrade

---

## 📖 About the Upgrade

The original project was a **CMD-based application** as required in the academic syllabus.
To push my learning further, I redesigned it with a **Graphical User Interface (GUI)** using Java Swing, keeping all backend logic and fundamentals unchanged.

With guidance from tools like ChatGPT, Claude, and Gemini, I improved the UI and made it look like a real desktop application that is secure, lightweight, and independent of external databases or internet connectivity.

---

## 📂 How to Run

1. Clone the repository

   ```bash
   git clone https://github.com/your-username/ReportGen.git
   ```
2. Open the project in any Java IDE (IntelliJ, Eclipse, NetBeans)
3. Compile and run the main Java file
4. Use the GUI to manage student records
